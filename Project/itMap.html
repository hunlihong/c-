Computer Science

A computer science major is a degree program that focuses on the theoretical and practical aspects of computing. Computer science majors learn about computer hardware, software, and algorithms. They also learn about the principles of problem-solving and how to design and implement solutions using computers. A computer science major can be a great choice for people who are interested in technology, problem-solving, and creativity. Computer science majors have a variety of career options available to them, including software development, data science, artificial intelligence, and cybersecurity.

Software Engineering/Development

Software engineering is the application of engineering principles to the design, development, testing, and maintenance of software. It is a broad field that encompasses many different aspects of software development, including:

  • Requirements engineering: Identifying the needs of the software users and stakeholders.
  • Software design: Creating a blueprint for the software system.
  • Software development: Coding the software system.
  • Software testing: Finding and fixing errors in the software system.
  • Software maintenance: Keeping the software system up-to-date and fixing bugs.
